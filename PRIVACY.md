# 개인정보 처리방침 (Privacy Policy)

**최종 업데이트**: 2026년 6월 4일

핏로그(FitLog, 이하 "본 앱")는 사용자의 개인정보를 소중히 여기며, 모든 데이터를 사용자의 디바이스에만 저장합니다.

## 1. 수집하는 정보

본 앱은 **어떠한 개인정보도 외부 서버로 전송하지 않습니다**. 다음 정보는 모두 디바이스의 로컬 SQLite 데이터베이스에만 저장됩니다:

- **건강 데이터** (HealthKit 연동): 수면, 심박, HRV, 운동 기록
- **위치 데이터** (GPS): 러닝 세션 중 경로 추적용
- **카메라/사진** (인바디 OCR): 인바디 결과지 이미지 — 텍스트 인식 후 즉시 메모리에서 폐기됨
- **모션 데이터** (CoreMotion): 농구 세션 중 점프·스프린트 감지용
- **사용자 입력 데이터**: 이름, 나이, 성별, 목표 시간, 신발, 보충제 등

## 2. 정보의 사용

수집된 데이터는 오직 본 앱 내에서 다음 목적으로 사용됩니다:

- 훈련 준비 점수 계산
- 운동 분석 및 추천
- 모닝 리포트 알림
- 데이터 시각화 (차트, 그래프)

## 3. 정보의 공유

본 앱은 **수집한 어떠한 정보도 제3자와 공유하거나, 외부 서버로 전송하거나, 판매하지 않습니다**. 모든 데이터 처리는 디바이스 내에서 완료됩니다.

## 4. 외부 서비스

본 앱은 다음 공개 API를 호출합니다 (개인 식별 정보 전송 안 함):

- **Open-Meteo API** (날씨 정보): 현재 위치의 위도·경도만 전송 (계정 정보 없음)

## 5. 데이터 보관 및 삭제

- 데이터는 사용자가 앱을 삭제할 때까지 디바이스에만 저장됩니다
- 앱 삭제 시 모든 데이터가 자동으로 삭제됩니다
- 본 앱 설정 → 데이터 백업 메뉴에서 사용자가 직접 JSON 파일로 백업·복원할 수 있습니다 (사용자 책임 하 관리)

## 6. 어린이 개인정보

본 앱은 9세 이상 사용자를 대상으로 합니다. 13세 미만 어린이로부터 의도적으로 개인정보를 수집하지 않습니다.

## 7. 권한 사용 안내

본 앱이 요청하는 모든 시스템 권한은 명확한 목적이 있습니다:

| 권한 | 목적 |
|---|---|
| HealthKit | 수면·심박·HRV 데이터를 읽어 훈련 준비 점수 계산 |
| 위치 (사용 중) | 현재 위치 날씨 조회 |
| 위치 (항상) | 러닝 중 화면 꺼져도 GPS 트래킹 (사용자가 명시적으로 허용한 경우만) |
| 카메라 | 인바디 결과지 촬영 (즉시 텍스트 추출 후 이미지 폐기) |
| 사진 라이브러리 | 인바디 결과지 이미지 선택 |
| 동작 및 피트니스 | 농구 세션 점프·스프린트 자동 카운트 |
| 알림 | 보충제 복용 시간, 모닝 리포트, 신발 교체 알림 |
